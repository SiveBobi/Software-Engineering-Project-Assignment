# Tic-Tac-Toe Command Line Game

## Project Description
This is a simple command-line Tic-Tac-Toe game developed in Python. 
The game allows two players to play against each other in the terminal.

This project was developed as part of the Postgraduate Diploma in Software Engineering assignment to demonstrate GitHub version control and documentation skills.

---

## Features
- Two-player gameplay (Player X and Player O)
- Input validation
- Winner detection
- Draw detection
- Clean terminal board display

---

## Requirements
- Python 3.x

## Version Control Strategy

This project was developed incrementally using meaningful Git commits to demonstrate proper version control practices.

Instead of uploading everything at once, the development process included separate commits for:

- Initial project setup
- Board display implementation
- Winner detection logic
- Input validation improvements
- README documentation
- Screenshot uploads
- Final formatting and cleanup

This demonstrates understanding of Git workflows and structured software development practices.
